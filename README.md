### 4. Skincare Registration Form ✨
A beautifully designed skincare profile registration form with full validation.

**Features:**
- Animated slide-in form with purple gradient background
- **Personal Information** section — Name, Email, Phone, Password, Date, City dropdown
- **Skin Profile** section — Skin type via radio buttons (Oily, Dry, Combination, Normal)
- **Monthly Budget** slider with live value display ($0–$100)
- **Products & Interests** checkboxes — Cleanser, Serum, Moisturizer, Sunscreen, Face Mask
- **Skin Concerns** checkboxes — Acne, Pigmentation, Fine Lines, Sensitive Skin
- Additional comments textarea
- Real-time form validation with red error highlighting
- Success message on valid submission
- Reset button to clear all fields
- Fully responsive for mobile screens
- simple design as for a skincare contact us page 
---

### 5. Skincare Brand Front Page 🌿
A complete landing page for a skincare brand built using HTML tables, Flexbox, and CSS.

**Features:**
- **Navbar** — Home, About, and Contact Us navigation links with hover scale effect
- **Cover Image** — Full-width brand banner with rounded bottom edges
- **Main Table Layout** — Three-column structured section containing:
  - **Main Menu** — Ordered list of product categories (Sunscreen, Serums, Facewash, Scrub)
  - **Our Goal** — Brand mission statement about safe, effective, and eco-friendly skincare
  - **Offers Panel** — Promotional banners (Flat 20% off on serums, First 20 customers offer, brand tagline)
- **Product Cards Section** — Responsive Flexbox grid showcasing 4 products:
  - Face Toner, Double Cleanser, Serum, Moisturizer
- Smooth card hover animation (lift and scale effect)
- Soft gradient background throughout the page
- Mobile responsive — product cards stack vertically on small screens
- Linked to the Skincare Registration Form via Contact Us

---

### 6. Web Car Showcase 🚗
A fully animated interactive car showcase page built purely with HTML, CSS, and JavaScript — no libraries or frameworks.

**Features:**
- **Night sky scene** — Deep blue gradient background with 120 randomly generated twinkling stars using CSS animations
- **City silhouette** — Procedurally generated buildings with glowing window grid patterns
- **Detailed road scene** — Asphalt road with yellow center dashes, white lane markings, sidewalk, curb, and road texture overlay
- **Street lamp posts** — Dynamically generated lamp posts with glowing yellow light halos across the full viewport width
- **Pure CSS car** — Fully detailed red sports car built entirely from HTML divs and CSS including:
  - Car body, roof, hood line, and bumper
  - Tinted glass windows with shine reflections
  - Door panels with handles
  - Headlights with glow effect
  - Tail lights with red glow
  - Spinning wheels with spokes, hub, tire tread, and underbody glow
- **Hover to race** — Hovering over the car triggers a smooth drive animation across the full screen width
- **Double-click to reset** — Car resets back to the starting position
- **Wheel spin animation** — Wheels spin using `requestAnimationFrame` only while the car is moving
- Google Fonts integration — Bebas Neue and Rajdhani typefaces
- Fully responsive layout
